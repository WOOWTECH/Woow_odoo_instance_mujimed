## 1. 平台總覽與帳號說明

本章說明沐淨醫美 Mujimed Odoo 18 醫美平台的整體架構，包含系統網址、預設帳號、醫護人員編制，以及平台角色與權限劃分。閱讀本章後，您將了解如何進入各系統前台與後台，以及各帳號的適用範圍。

### 步驟 1：系統網址一覽

沐淨醫美平台採用 Odoo 18 多網站（Multi-Website）架構，同時營運 B2C 消費者網站與 B2B 企業批發網站，並共用同一個後台管理介面。

| 系統 | 網址 | 說明 |
|------|------|------|
| B2C 消費者網站 | https://b2cmujimed-odoo.woowtech.io | 面向一般消費者的醫美療程預約與購物平台 |
| B2B 企業網站 | https://b2bmujimed-odoo.woowtech.io | 面向診所、醫美業者的企業批發採購平台 |
| 後台管理介面 | https://b2cmujimed-odoo.woowtech.io/odoo | Odoo 18 後台管理系統入口 |

> **提示**：B2C 與 B2B 網站共用同一個 Odoo 資料庫，後台管理員可以在同一介面中管理兩個網站的內容、商品與訂單。

### 步驟 2：預設帳號與角色

系統預設提供三種角色帳號，分別對應不同的使用情境：

| 帳號 | 密碼 | 姓名 | 角色 | 適用網站 |
|------|------|------|------|---------|
| `admin` | `admin` | 管理員 | 全系統管理 | 後台管理（`/odoo`） |
| `portal` | `portal` | 王小美 | B2C 會員 | B2C 消費者網站 |
| `partner` | `partner` | 美麗佳人診所 | B2B 企業夥伴 | B2B 企業網站 |

**角色說明：**

- **管理員（admin）**：擁有 Odoo 全系統最高權限，可管理網站內容、商品目錄、銷售訂單、預約排班、忠誠計畫、使用者權限等所有模組。
- **B2C 會員（portal）**：一般消費者帳號，可在 B2C 網站上瀏覽療程、線上預約、購買療程、查看訂單與發票、管理會員卡。
- **B2B 企業夥伴（partner）**：企業採購帳號，可在 B2B 網站上瀏覽批發商品、下單採購、管理企業訂單與發票。

沐淨醫美配置專業醫師與護理師團隊，以下為系統內建的醫護人員資料：

| 王建國醫師 | `dr.wang` | 體雕抗老 | 負責冷凍溶脂、電波拉皮、音波拉提等體雕抗老療程 |

### 步驟 4：平台架構說明

沐淨醫美平台基於 Odoo 18 建置，整合以下核心模組：

| 功能模組 | 說明 |
|---------|------|
| 網站（Website） | 多網站架構，同時營運 B2C 與 B2B 前台 |
| 電子商務（eCommerce） | 線上療程商城與企業批發商城 |

| 發票（Invoicing） | 發票開立、收款管理 |
| 忠誠計畫（Loyalty） | 電子錢包、忠誠點數、禮品卡、優惠券、寄品卡 |

| Portal | 會員自助入口，查看訂單、發票、預約、會員卡 |

## 2. B2C 首頁與品牌形象

B2C 消費者網站首頁是沐淨醫美面對一般消費者的第一印象。首頁採用現代化醫美品牌設計，由上而下依序呈現品牌主視覺、療程服務區塊、信任指標、行動呼籲橫幅，以及聯絡資訊與頁尾。本章將逐一介紹各區塊的內容與功能。

### 步驟 1：品牌主視覺（Hero Section）

![B2C 首頁品牌主視覺](ch02_01_b2c_hero.png)

首頁最上方為全寬品牌主視覺區塊，包含以下元素：

| 功能 | 說明 |
|------|------|
| 品牌標題 | 顯示「沐淨醫美」品牌名稱與品牌標語 |
| 主視覺圖片 | 高品質醫美形象照片，傳遞專業與美感 |
| 行動呼籲按鈕 | 引導訪客進入療程商城或線上預約 |
| 導覽列（Navbar） | 頂部固定導覽列，包含首頁、療程商城、預約掛號、聯絡我們等連結 |
| 會員登入 | 導覽列右上角提供登入 / 註冊入口 |

1. 開啟瀏覽器，前往 `https://b2cmujimed-odoo.woowtech.io`。
2. 頁面載入後，首先映入眼簾的即為品牌主視覺區塊。
3. 點擊導覽列中的各連結，可前往對應功能頁面。
4. 點擊右上角「登入」按鈕，可進入會員登入頁面。

### 步驟 2：療程服務區塊（Services Section）

![B2C 療程服務區塊](ch02_02_b2c_services.png)

品牌主視覺下方為療程服務區塊，以卡片式排列展示沐淨醫美提供的核心療程分類：

| 功能 | 說明 |
|------|------|
| 療程分類卡片 | 每張卡片代表一個療程分類，搭配圖示與簡介 |
| 分類名稱 | 如微整形注射、光電雷射、體雕療程、臉部保養、特殊療程 |
| 分類圖示 | 專屬圖示設計，幫助訪客快速辨識療程類型 |
| 查看更多連結 | 點擊卡片或連結可前往該分類的療程列表頁面 |

1. 向下捲動首頁，即可看到療程服務區塊。
2. 瀏覽各療程分類卡片，了解沐淨醫美提供的服務項目。
3. 點擊有興趣的分類卡片，將跳轉至對應的療程商城頁面。

### 步驟 3：信任指標區塊（Trust Section）

![B2C 信任指標區塊](ch02_03_b2c_trust.png)

信任指標區塊透過數據與認證資訊，建立訪客對品牌的信心：

| 功能 | 說明 |
|------|------|
| 服務人數 | 顯示累計服務人數或療程次數 |
| 專業認證 | 展示診所相關醫療認證與合格證書 |
| 顧客滿意度 | 顧客好評率或滿意度數據 |
| 醫師團隊 | 專業醫師人數或團隊介紹 |

1. 繼續向下捲動頁面，即可看到信任指標區塊。
2. 此區塊以視覺化方式呈現關鍵信任數據。
3. 訪客可藉此評估品牌的專業度與可信度。

### 步驟 4：行動呼籲橫幅（CTA Banner）

![B2C 行動呼籲橫幅](ch02_04_b2c_cta_banner.png)

行動呼籲橫幅以醒目的設計鼓勵訪客立即採取行動：

![B2C 聯絡資訊與頁尾](ch02_05_b2c_contact_footer.png)

| 聯絡電話 | 診所聯絡電話號碼 |

## 3. B2B 首頁與企業入口

B2B 企業網站面向醫美診所、醫療機構及企業夥伴，提供醫美耗材與設備的批發採購服務。本章介紹 B2B 首頁的各區塊設計與功能。

### 步驟 1：B2B 品牌主視覺（Hero Section）

![B2B 首頁品牌主視覺](ch03_01_b2b_hero.png)

B2B 首頁最上方為企業形象主視覺區塊：

| 功能 | 說明 |
|------|------|
| 企業標題 | 顯示沐淨醫美 B2B 批發平台名稱與標語 |
| 主視覺圖片 | 醫療設備或專業耗材形象照片 |
| 企業登入按鈕 | 引導企業夥伴登入或註冊 |
| 商品目錄按鈕 | 直接前往 B2B 批發商品目錄 |
| 導覽列 | 頂部固定導覽列，含首頁、商品目錄、聯絡我們等連結 |

1. 開啟瀏覽器，前往 `https://b2bmujimed-odoo.woowtech.io`。
2. B2B 首頁主視覺以專業企業形象為主軸，與 B2C 消費者網站風格有所區隔。

| 體雕設備耗材 | 電波、音波、冷凍溶脂等設備耗材 |
| 保養品 | 術後修護、精華液等專業保養品 |

| 診所用品 | 手套、紗布等一般診所消耗品 |

1. 向下捲動 B2B 首頁，可看到商品分類區塊。
2. 點擊任一分類卡片，將跳轉至該分類的商品列表頁面。
3. 每個分類下方會顯示該分類的商品數量。

5. 付款成功後，系統將顯示訂單確認頁面。

## 9. B2C 預約掛號 — 日曆選時與確認

選擇療程與醫師後，接下來需要在日曆上選擇可用的時段，並填寫預約資訊完成預約。本章說明日曆選時與預約確認的完整流程。

### 步驟 1：日曆選擇預約時段

![日曆與時段選擇](ch09_01_calendar_or_staff.png)

進入日曆選時步驟後，頁面將顯示可用的預約時段：

| 功能 | 說明 |
|------|------|
| 月曆視圖 | 以月曆形式顯示可預約的日期 |
| 可用日期標示 | 有可用時段的日期會以特殊色彩標示 |
| 不可用日期 | 已額滿或休診日期將被灰色標示或不可點選 |
| 時段列表 | 點選日期後，顯示該日期的可用時段 |
| 時段按鈕 | 每個可用時段以按鈕形式顯示，點擊即可選擇 |
| 已選時段 | 選擇後以醒目色彩標示已選擇的時段 |

1. 在月曆上瀏覽可用的預約日期。
2. 點擊您希望預約的日期。
3. 系統將顯示該日期的所有可用時段。
4. 點擊您偏好的時段按鈕。
5. 確認日期與時間無誤後，系統將進入預約表單填寫步驟。

### 步驟 2：填寫預約資訊

選擇時段後，需填寫預約相關資訊：

| 功能 | 說明 |
|------|------|
| 姓名 | 預約者姓名（已登入會自動帶入） |
| 電子郵件 | 聯絡用電子郵件（已登入會自動帶入） |
| 電話號碼 | 聯絡用電話號碼 |
| 備註說明 | 可填寫特殊需求或補充說明 |
| 預約摘要 | 顯示已選擇的療程、醫師、日期、時段 |
| 確認預約按鈕 | 點擊送出預約 |

1. 確認或填寫個人聯絡資訊。
2. 如有特殊需求，可在備註欄位中說明。
3. 仔細核對預約摘要中的療程、醫師、日期與時段是否正確。

2. 建議下載 ICS 檔案加入行事曆，以免忘記預約時間。

## 28. 常見問題（FAQ）

本章彙整使用沐淨醫美 Odoo 18 醫美平台時常見的問題與解答。

### Q1：如何登入 B2C 消費者網站？

**A**：前往 `https://b2cmujimed-odoo.woowtech.io`，點擊右上角的「登入」按鈕，輸入您的帳號與密碼即可登入。示範帳號為 `portal` / `portal`（王小美）。

### Q2：如何登入 B2B 企業網站？

**A**：前往 `https://b2bmujimed-odoo.woowtech.io`，點擊「登入」按鈕，使用企業夥伴帳號登入。示範帳號為 `partner` / `partner`（美麗佳人診所）。

### Q3：如何進入後台管理介面？

**A**：前往 `https://b2cmujimed-odoo.woowtech.io/odoo`，使用管理員帳號 `admin` / `admin` 登入即可進入 Odoo 18 後台管理系統。

### Q4：忘記密碼怎麼辦？

**A**：在登入頁面點擊「忘記密碼？」連結，輸入您的註冊信箱，系統將發送密碼重設郵件。請依郵件中的指示設定新密碼。

### Q5：如何註冊新帳號？

**A**：在登入頁面點擊「立即註冊」連結，填寫姓名、Email 與密碼等資訊，完成註冊後即可使用新帳號登入。

### Q6：B2C 網站提供哪些療程？

**A**：B2C 網站共提供 20 種療程，分為五大分類：
- **微整形注射**（4 項）：玻尿酸填充、肉毒桿菌注射、膠原蛋白增生劑、消脂針
- **光電雷射**（5 項）：皮秒雷射、飛梭雷射、淨膚雷射、脈衝光 IPL、除毛雷射
- **體雕療程**（3 項）：冷凍溶脂、電波拉皮 Thermage、音波拉提 HIFU
- **臉部保養**（4 項）：水光注射、導入保養、化學煥膚、杏仁酸煥膚
- **特殊療程**（4 項）：PRP 自體血小板、雷射除斑、痘疤治療、初次諮詢

### Q7：如何預約療程？

**A**：登入 B2C 網站後，點擊導覽列的「預約掛號」，選擇療程類型，選擇醫師（或不指定），在日曆上選擇日期與時段，填寫預約資訊後送出即完成預約。

### Q8：如何查看我的預約記錄？

**A**：登入後前往 Portal 儀表板（`/my`），點擊「我的預約」功能卡片，即可查看所有預約記錄。

### Q9：可以取消預約嗎？

**A**：若預約狀態為「已確認」且尚未執行，您可以在預約詳情頁面中點擊「取消預約」按鈕進行取消。已完成的預約無法取消。實際取消政策依診所規定。

### Q10：如何使用購物車購買療程？

**A**：在療程商城中將想要的療程加入購物車，點擊購物車圖示查看購物車內容，確認無誤後點擊「結帳」，填寫結帳資訊並完成付款。

### Q11：如何使用優惠券或優惠碼？

**A**：在購物車頁面中找到「優惠碼」輸入欄位，輸入您的優惠碼後點擊「套用」，系統將自動計算折扣並更新合計金額。

### Q12：如何查看我的會員卡餘額？

**A**：登入後前往 Portal 儀表板，點擊「會員卡」功能卡片，即可查看所有會員卡的餘額與點數。

### Q13：什麼是電子錢包？

**A**：電子錢包是沐淨醫美提供的儲值型忠誠計畫。您的錢包中的點數可在結帳時直接折抵消費金額，1 點 = 1 元。王小美帳號預設有 5,000 點餘額。

### Q14：什麼是忠誠點數？

**A**：忠誠點數是消費自動累積的點數。每次消費可依規則累積點數，累積到一定數量可兌換療程折扣或贈品。王小美帳號預設有 1,200 點。

### Q15：什麼是醫美寄品卡？

**A**：醫美寄品卡允許您預購特定療程的次數，寄存於診所後可分次使用。例如購買 10 次導入保養，每次到診使用 1 次，直到用完為止。

### Q16：B2B 網站有哪些商品？

**A**：B2B 企業批發網站提供 31 項商品，分為 6 大分類：注射耗材（5 項）、雷射耗材（5 項）、體雕設備耗材（5 項）、保養品（5 項）、儀器配件（5 項）、診所用品（6 項）。

### Q17：如何成為 B2B 企業夥伴？

**A**：請透過 B2B 網站的「聯絡我們」頁面提交企業合作申請，或直接聯繫沐淨醫美業務團隊。經審核通過後，將為您建立企業夥伴帳號。

### Q18：B2B 企業採購有什麼優勢？

**A**：B2B 企業夥伴享有批發優惠價格、原廠正品認證、快速出貨服務、專屬客服支援，以及完善的企業帳務與發票管理功能。

### Q19：手機可以正常使用網站嗎？

**A**：是的，沐淨醫美平台採用響應式設計，完整支援手機與平板裝置。所有功能（瀏覽、預約、購物、Portal 管理）均可在行動裝置上正常操作。

### Q20：後台管理有哪些主要功能？

**A**：後台管理（Odoo 18 管理介面）提供以下主要功能：
- 多網站管理（B2C + B2B）
- 產品目錄管理（療程 + 批發商品）
- 銷售訂單管理
- 發票與帳務管理
- 預約與排班管理
- 忠誠計畫與會員卡管理
- 客戶與合作夥伴管理

### Q21：如何在後台新增產品？

**A**：登入後台後，進入「產品」模組，點擊「新增」按鈕，填寫產品名稱、類型、價格、描述等資訊，設定適用網站後儲存。將發佈狀態設為「已發佈」即可在前台網站上顯示。

### Q22：如何在後台管理預約？

**A**：登入後台後，進入「預約」模組，可查看所有預約記錄。可使用行事曆視圖查看排程，點擊個別預約可變更狀態（確認、完成、取消）。

### Q23：如何為客戶發放會員卡？

**A**：登入後台後，進入「忠誠計畫」模組，選擇對應的忠誠計畫，在「會員卡」頁籤中點擊「新增」，選擇客戶並設定初始餘額或點數，儲存後即完成發放。

### Q24：系統支援哪些付款方式？

**A**：系統支援的付款方式依後台設定而定，可能包含線上信用卡付款、銀行轉帳、現場付款等。B2B 企業客戶可能享有月結付款等企業專屬方式。具體付款方式請參閱結帳頁面的選項。

### Q25：網站出現錯誤或無法載入怎麼辦？

**A**：請嘗試以下步驟：
1. 清除瀏覽器快取，重新整理頁面。
2. 嘗試使用其他瀏覽器或無痕模式開啟。
3. 確認網路連線是否正常。
4. 若問題持續，請透過「聯絡我們」頁面回報問題，或聯繫沐淨醫美客服團隊。

### Q26：如何修改個人資料？

**A**：登入後前往 Portal 儀表板，點擊「個人資料」功能卡片（或前往 `/my/account`），即可修改姓名、電話、地址等個人資訊。修改完成後點擊「儲存」。

### Q27：如何修改密碼？

**A**：登入後前往 Portal 儀表板，點擊「安全性」功能卡片（或前往 `/my/security`），輸入目前密碼與新密碼，點擊「確認」即可完成密碼修改。

### Q28：發票可以下載 PDF 嗎？

**A**：可以。前往「我的發票」頁面，點擊要下載的發票，在發票詳情頁面中點擊「下載 PDF」按鈕即可下載發票 PDF 檔案。

### Q29：預約的療程和商城購買的療程有什麼不同？

**A**：預約功能用於預約到診時間，確保您到院時有醫師為您服務。商城購買功能用於線上付款購買療程。建議先透過商城購買療程取得訂單，再透過預約功能預約到診時間。

### Q30：如何聯絡沐淨醫美？

**A**：您可以透過以下方式聯繫沐淨醫美：
1. 前往 B2C 網站的「聯絡我們」頁面，填寫聯絡表單。
2. 撥打網站上公佈的診所聯絡電話。
3. 親臨沐淨醫美診所。
